Calculator App | Android Studio + Flutter

Project Overview:

This is a simple and responsive Calculator application built using Flutter.
The app performs basic arithmetic operations with a clean and user-friendly interface.

The project demonstrates:

* Basic mobile app development using Flutter
* UI design using Material Design
* Handling user input and calculations
* Building responsive mobile interfaces


Features:

* Perform basic arithmetic operations (+, −, ×, ÷)
* Clear (AC) button to reset calculations
* Real-time calculation display
* Responsive and clean UI design
* Error handling (like division by zero)



Technologies Used:

* Flutter
* Dart
* Material Design


Project Structure:
lib folder contains main app code
main.dart is the entry point of the application
widgets folder contains reusable UI components (if used)
assets folder contains images or icons (if used)
pubspec.yaml contains dependencies and configuration


How to Run:

1. Install Flutter SDK
2. Clone or download the project
3. Run flutter pub get to install dependencies
4. Run flutter run to start the app
5. Use an emulator or physical device to view the app
